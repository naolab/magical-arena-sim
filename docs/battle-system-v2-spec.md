## 4すくみシステム

### 4つの感情アクション

1. **Rage（激怒）**
2. **Terror（恐怖）**
3. **Grief（悲嘆）**
4. **Ecstasy（恍惚）**

### 相性関係

```
Rage（激怒） > Terror（恐怖） > Grief（悲嘆） > Ecstasy（恍惚） > Rage
```

- **Rage > Terror**: 激怒が恐怖を圧倒する
- **Terror > Grief**: 恐怖が悲嘆を麻痺させる
- **Grief > Ecstasy**: 悲嘆が恍惚を現実に引き戻す
- **Ecstasy > Rage**: 至福が怒りを溶かす
