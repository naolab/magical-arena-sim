# 🎮 バトルシステム仕様書（Ver.1.2）

---

## 🧩 基本概要

- コマンド選択型・ターン制バトル。
- 各ターンで以下の 3 択から行動を選択する。
  - **アタック (Attack)**
  - **アピール (Appeal)**
  - **ガード (Guard)**
- 三すくみ構造:
  - アタック ＞ アピール
  - アピール ＞ ガード
  - ガード ＞ アタック

### 勝敗結果

| 結果 | 効果 |
|------|------|
| 勝利 | 技発動 + ファン獲得 + アンチ軽減 |
| 引き分け | ファンの多い方が有利 |
| 敗北 | 被弾 + アンチ上昇 |

---

## ⚔️ 技の種類と性能

| 技名 | 勝利対象 | 技倍率 | ダメージ特性 | ファン変動 | アンチ変動 | 特徴・バランス意図 |
|------|----------|--------|--------------|------------|------------|----------------------|
| アタック (Attack) | アピール | ×1.2 | 通常攻撃 (高火力) | ±0% | +5 | 攻撃の主軸。連発するとアンチが上昇。 |
| アピール (Appeal) | ガード | ×0.0 | ダメージなし | +10% | −15 | ファンを大幅に獲得し、人気を回復。 |
| ガード (Guard) | アタック | ×0.0 | 被ダメージ 80% カット (1/5) | +3% | −5 | 防御しながら少しだけファンを獲得。 |

### アタック (Attack)

- 高火力の主攻撃手段。
- 三すくみでアピールに勝利。
- ファン補正が高いほど威力が上昇。
- 連発しすぎるとアンチが蓄積 (+5〜10)。
- 観客の支持を得られなければ炎上リスク。

### アピール (Appeal)

- ファンを増やす行動で、ダメージは発生しない。
- 成功時にファン +10%、アンチ −15。
- ファン増加が次ターン以降の火力上昇につながる。
- 観客指示「アピールして！」で最大効果を発揮。

### ガード (Guard)

- 防御行動でアタックに強い。
- 被ダメージを 80% カット (実質 1/5)。
- ファン +3%、アンチ −5 と冷静な対応が評価される。
- 守りながら魅せることで選択価値を確保。

---

## 🧠 ターン進行フロー

1. 観客の指示発表
2. プレイヤーと敵の行動選択
3. 勝敗判定 (三すくみロジック)
4. アンチゲージ変動 (プレイヤーのみ)
5. ダメージ計算と HP 更新
6. ファン変動処理
7. アンチゲージ判定（Lv に応じてペナルティ適用）
8. 次ターン移行

## 🎤 オーディエンスシンクシステム (Audience Sync)

### 観客構成

- 🟥 相手のファン (Enemy Fans)
- 🟨 どっちつかずのファン (Neutral Fans)
- 🟩 自分のファン (Your Fans)

初期構成例: 相手 20% / どっちつかず 70% / 自分 10%

### ダメージ計算式

```
最終ダメージ = 基礎火力 × 技倍率 × (1 + 自分ファン率)
```

| ファン率 | 火力補正 |
|----------|----------|
| 0〜20% | ×1.0 |
| 21〜50% | ×1.2 |
| 51〜80% | ×1.5 |
| 81〜100% | ×2.0 |

### ファン変動式

```
Δ自分ファン = 勝敗補正 + 指示補正 − アンチ補正
```

| 種類 | 条件 | 変化量 | 備考 |
|------|------|---------|------|
| 勝敗補正 | 勝利 | +10% | 相手 −5%、どっちつかず −5% |
| 勝敗補正 | 引き分け勝利 | +5% | 小幅増加 |
| 勝敗補正 | 敗北 | −5% | 自分 → どっちつかず |
| 指示補正 | 指示に従う | +5% | どっちつかず → 自分 |
| 指示補正 | 指示に反する | −5% | 自分 → どっちつかず |
| アンチ補正 | Lv1 | −3% | 緩やかな人気低下 |
| アンチ補正 | Lv2 | −6% | 明確な人気低下 |
| アンチ補正 | Lv3 | −10% | 炎上状態・継続減少 |

## 👂 観客の指示システム (Audience Command)

### 指示例と効果

| 指示 | 成功時効果 | 失敗時効果 |
|------|------------|------------|
| 攻撃しろ！ | アタック選択でファン +5% | アピール・ガード選択でアンチ +10 |
| アピールして！ | アピール選択でファン +5% | 他行動でアンチ +10 |
| ガードするな！ | ガード以外でファン +5% | ガード選択でアンチ +15 |

### 指示出現確率 (暫定)

| 指示タイプ | 確率 |
|------------|------|
| 攻撃しろ！ | 40% |
| アピールして！ | 30% |
| ガードするな！ | 30% |

## 😈 アンチゲージシステム (Anti Gauge)

- プレイヤー専用ゲージ。
- 観客の期待を裏切る行為で上昇。
- 高まるほど人気が低下し、最悪ブーイングが発生。

| Lv | 範囲 | ファン獲得補正 | 火力補正 | 状態演出 |
|----|------|-----------------|-------------|--------------|
| Lv0 | 0〜29 | 100%（ペナルティなし） | 100% | 通常状態 |
| Lv1 | 30〜59 | 80%（−20%） | 90%（−10%） | 観客が冷め始める |
| Lv2 | 60〜89 | 60%（−40%） | 75%（−25%） | 明確なブーイング |
| Lv3 | 90〜100 | 40%（−60%） | 60%（−40%） | 炎上状態・大幅な弱体化 |

- Lv が上がるほどファン獲得量と与ダメージが大きく減少する。
- Lv3（炎上状態）でも行動制限はないが、ファン・火力のペナルティが最も重い。

## 🎯 デザイン上の狙い

- 人気と火力を連動させ、観客との心理戦を重視。
- アピールは攻めへの布石、ガードは演出として機能。
- 攻め・守り・魅せが三すくみで成立。
- アンチゲージでリスク管理を導入し、単調なプレイを防止。
